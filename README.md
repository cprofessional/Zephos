# Zephos  

Zephos is an AI assistant built with Next.js and TypeScript. It uses the latest OpenAI model, further trained on personal content.  

## Tech Stack  
- Next.js  
- TypeScript  
- OpenAI API  

## Features  
- Custom-trained AI responses  
- Fast and scalable  
- Adaptive learning

## Requirements 
- Implement NEXT_PUBLIC_OPENAI_KEY in .env
